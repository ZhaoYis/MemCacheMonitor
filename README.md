# MemCacheMonitor

项目提供了一个MemCache的监控程序，网上提供的大部分都是Php写的工具（memadmin 是一款不错的监控工具），需要手动部署，操作不是很方便，因此有了这一款桌面
程序，感谢博客园小伙伴（晓风拂月）提供源码，文章地址：http://www.cnblogs.com/xffy1028/archive/2013/02/01/2861706.html，我在此基础上做了一些小的小调整，
修复了一些Bug。

### 项目基本实现的功能：
1、显示MemCache 基本信息
2、监控统计MemCache 信息
3、对MemCache 的区块进行统计
4、MemCache 中的数据维护
5、计算各个服务器的命中率
6、对服务器列表的监控
7、服务器数据备份
8、服务器数据还原
9、添加服务器
